# 🎬 Movie Recommendation System

This project is a **Movie Recommendation System** built using **Python and Data Science concepts**.  
It recommends movies to users based on their ratings and similarity between movies.

## 📌 Features
- User-based movie recommendations  
- Uses movie rating dataset (`user_movie_rating.csv`)
- Simple and beginner-friendly implementation
- Clean and structured project files

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit (for UI)

## 📂 Dataset
- `user_movie_rating.csv`  
  - Contains user ratings for different movies

## ▶️ How to Run
1. Clone the repository  
2. Install required libraries 
3. Run the application  


## 🎯 Learning Outcome
- Understanding recommendation systems  
- Working with real datasets  
- Data preprocessing and similarity concepts  
- Basic ML project structure
